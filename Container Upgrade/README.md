# Container Home Skill

A Claude skill that holds all project context for Mick's off-grid shipping
container lake home at Land Between the Lakes, Dover, Tennessee.

## What This Skill Does

- Answers design and construction questions with full project context already loaded
- Assists with expansion planning and phase sequencing
- Helps navigate septic permitting and systems planning
- Generates Facebook posts about the project in Mick's voice
- Provides technical fabrication and welding guidance specific to container construction

## Trigger Phrases

This skill activates when you mention any of the following:

- Container home, lake home, container house
- Stewart County, LBL, Dover TN
- The expansion, adding containers
- Solar system, septic permitting
- Retirement property, geographic arbitrage
